# SenyaMatika - Mathematics Learning App

Project Overview

SenyaMatika is an innovative educational Flutter application designed to make learning mathematics fun and engaging. The app provides interactive lessons, quizzes, progress tracking, and sign language support to help students master mathematical concepts.

# Features

Core Features

- Interactive Learning Space: Access mathematics topics with visual content

- Sign Language Avatar: Real-time speech-to-sign language translation

- Progress Tracking: Monitor learning progress with detailed analytics

- Quiz System: Test knowledge with various quiz formats

- Sign Language Dictionary: Learn sign language for mathematical terms

# Topics Covered
- Number Values (Whole Numbers, Comparison, Ordinal Numbers, Money Value)

- Fundamental Operations (Addition, Subtraction, Multiplication, Division)

- Fraction (Basic concepts)

- Decimal Numbers (Basic concepts)

- Percentage (Concepts of percentage)

- Mensuration (Time, Days/Weeks/Months)

# SET Up Guide

Prerequisites
- Flutter SDK: Version 3.9.2 or higher

- Dart SDK: Version 3.x

- IDE: Android Studio, VS Code, or any Flutter-compatible IDE

- Device: Android emulator, or physical device
