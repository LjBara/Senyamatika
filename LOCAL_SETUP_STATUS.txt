🎯 NEXT STEPS TO START LOCAL TESTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Install PostgreSQL
   📥 Download: https://www.postgresql.org/download/windows/
   ⚙️  Install with default settings
   🔑 Set password for 'postgres' user (REMEMBER THIS!)
   ⏱️  Time: ~5 minutes

STEP 2: Create Database
   Option A (GUI):
      1. Open pgAdmin 4
      2. Right-click Databases → Create → Database
      3. Name: senyamatika
   
   Option B (Command):
      psql -U postgres -c "CREATE DATABASE senyamatika;"
   
   ⏱️  Time: ~1 minute

STEP 3: Update .env File
   📝 Edit: backend/.env (already open in your editor)
   🔄 Replace: YOUR_POSTGRES_PASSWORD
   ✏️  With: your actual PostgreSQL password
   
   Example:
   DATABASE_URL=postgresql://postgres:mypassword123@localhost:5432/senyamatika
   
   ⏱️  Time: ~30 seconds

STEP 4: Install Dependencies
   📂 cd backend
   📦 npm install
   ⏱️  Time: ~2 minutes

STEP 5: Start Backend Server
   🚀 npm start
   
   Expected output:
   🚀 Server running on port 3000
   ✅ Database connected successfully
   ✅ Database tables initialized successfully
   
   ⏱️  Time: ~10 seconds

STEP 6: Test API
   Open new terminal:
   
   # Test health
   curl http://localhost:3000/health
   
   # Test registration
   curl -X POST http://localhost:3000/api/auth/register ^
     -H "Content-Type: application/json" ^
     -d "{\"email\":\"test@test.com\",\"password\":\"pass123\",\"name\":\"Test\"}"
   
   ⏱️  Time: ~1 minute

STEP 7: Test with Flutter App
   1. Update lib/backend/services/api_service.dart
   2. Change baseUrl to: http://localhost:3000/api
   3. Run: flutter run -d chrome
   4. Test registration and login
   
   ⏱️  Time: ~5 minutes
